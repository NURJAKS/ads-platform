# Фронтенд платформы объявлений

React приложение для работы с платформой объявлений.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

## Переменные окружения

Создайте файл `.env` в корне проекта:

```
VITE_API_URL=http://localhost:8000/api/v1
```

## Структура проекта

- `src/api/` - API клиенты для взаимодействия с бекендом
- `src/components/` - Переиспользуемые компоненты
- `src/context/` - React контексты (AuthContext)
- `src/pages/` - Страницы приложения

## Основные функции

- Регистрация и авторизация пользователей
- Просмотр объявлений с фильтрацией
- Создание и редактирование объявлений
- Админ-панель для модерации объявлений
- Управление избранными объявлениями
